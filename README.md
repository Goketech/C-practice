# C Practice Updates

This repository contains a few beginner-friendly C programs I worked on while practicing the basics of programming. I wrote these updates as a software engineering student, so the focus was on learning, testing ideas, and getting comfortable with functions, conditionals, loops, and recursion.

## What Was Updated

- `1.c` now shows a simple addition example using two integer values and prints the result clearly.
- `2.c` was turned into a menu-based calculator that supports addition, subtraction, multiplication, and division.
- `3.c` explains recursion and demonstrates it with a factorial program.

## Notes On The Programs

- The calculator includes input validation and handles division by zero.
- The factorial program asks for a positive integer and uses a recursive function to compute the answer.
- The first example is a small starting point that helped me practice variables, arithmetic, and formatted output.

## How To Run

Compile any file with `gcc` and run the generated executable.

```bash
gcc 1.c -o 1
./1

gcc 2.c -o 2
./2

gcc 3.c -o 3
./3
```

## Closing Thought

These exercises helped me build confidence with C syntax and problem solving one step at a time.
